## Introduction
{Project Name} is a free and open-source script editing web application that does not now and will never require any payment, recurring subscription, personal information, or onerous terms of service to use.

To use {Project Name}, simply start writing, or upload an existing .FDX file to start editing. You can also download {Project Name} as a Progressive Web App in order to use it offline anywhere.

{Project Name} uses your browser's natively available localStorage in order to save your settings and scripts, though you are encouraged to download them as .FDX files as well. While {Project Name} will never *require* a log in of any kind, you may link your Google, Microsoft, or GitHub account to save your scripts there.

## Using {Project Name}
{Project Name} is purposefully designed to be familiar to experienced screenwriters while being accessible for newcomers to the craft. Its closest analogue is the Final Draft lineage of screenwriting softwares, and you can rely on your muscle memory from years of typing in that script editor when writing your screenplays in {Project Name}.
